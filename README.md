# SkyAuctions
A project for 4SA3. Uses REST API from Hypixel and SQL Server Database.

Config should be in config.cfg file in working directory.
```
[Database]
DRIVER = 
SERVER =
DATABASE = 
USER = 
PASSWORD = 
[Logging]
FILE_NAME = SkyAuctions.log
LOG = TRUE
DEBUG = FALSE
NORMAL = TRUE
ERROR = TRUE
OTHER = TRUE
```
